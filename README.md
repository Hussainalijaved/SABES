# ⚡ SABES – Smart Billing & Electrical Store Management System

SABES (Sami and Brothers Electric Store) is a modern mobile application built using .NET MAUI Blazor Hybrid, designed to streamline billing and inventory management for electrical stores. The application focuses on delivering a fast, offline-capable, and user-friendly billing experience with professional output generation.

---

## 🚀 Overview

SABES is tailored for small to medium-sized electrical businesses, enabling efficient management of products, categories, and billing operations. The system provides real-time calculations, flexible discount handling, and professional invoice generation, all within a mobile-first interface.

---

## ✨ Key Features

### 🧾 Billing System

* Add multiple items to a bill dynamically
* Automatic total calculation
* Apply discounts (percentage-based)
* Display total before and after discount
* Mark bills as:

  * Paid
  * Pending
  * Quotation

---

### 📦 Product & Category Management

* Manage product categories
* Add items with:

  * Price
  * Company
  * Electrical specifications

---

### 📄 Invoice Generation

* Generate professional PDF invoices
* Detailed itemization
* Branded layout for business use

---

### 📱 Mobile-First UI/UX

* Responsive design using Tailwind CSS
* Bottom navigation bar for easy access
* Clean and professional layout
* Icon-based actions for better usability

---

### 💾 Offline Capability

* Uses local SQLite database
* Works بدون internet
* Fast and reliable data access

---

## 🛠️ Tech Stack

* **Framework:** .NET MAUI Blazor Hybrid
* **UI:** Razor Components + Tailwind CSS
* **Database:** SQLite (local storage)
* **Architecture:** Component-based + MVVM pattern
* **PDF Generation:** Custom implementation for invoices

## ⚙️ Installation & Setup

1. Clone the repository:

```bash id="9qk5eh"
git clone https://github.com/your-username/SABES.git
```

2. Open in Visual Studio (ensure .NET MAUI workload is installed)

3. Restore dependencies:

```bash id="d3c6sx"
dotnet restore
```

4. Build and run:

```bash id="l3i9q1"
dotnet build
dotnet run
```
